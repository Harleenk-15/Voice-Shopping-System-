# Voice Shopping Assistant

A simple and interactive **Voice-Controlled Shopping Assistant** built using **HTML, CSS, and JavaScript**.
This web application allows users to add or remove shopping items using voice commands, automatically calculates the total cost, and stores the shopping list in the browser.

---

## Features

* Voice command input using browser speech recognition
* Add items with price using voice
* Remove items from the list
* Automatic total price calculation
* Category detection (dairy, fruits, grains, etc.)
* Persistent storage using localStorage
* Clean and responsive user interface
* Dynamic background changes for better user experience



## Technologies Used

* HTML
* CSS
* JavaScript
* Web Speech API (Speech Recognition & Speech Synthesis)
* Browser Local Storage



## How to Use

1. Open the `index.html` file in Google Chrome.
2. Click the **Speak** button.
3. Say commands like:

   * “Add milk 40”
   * “Add rice 60”
   * “Remove milk”
4. The item will appear in the list with its price.
5. The total will update automatically.



## Example Voice Commands

* Add apple 30
* Add bread 25
* Remove apple
* Clear list (using the button)
